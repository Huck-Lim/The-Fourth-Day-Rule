# 初四法则 · 自由消费计算器

## 项目简介

这是一个帮助你理性消费的工具。它源自一个观察：过度节俭是对自己时间的贱卖，而无节制消费则是对未来的透支。“初四法则”通过两个简单的公式，帮你量化时间价值与消费能力，在“省小钱”与“买快乐”之间找到平衡。

---

## 一、为什么研究这个？

我们从小被教育“节俭是美德”，但过度节俭其实是对自己时间的贱卖。

回老家，常见长辈为省一两块钱折腾半小时，或“没苦硬吃”——过期东西舍不得扔，亲力亲为却耗时费力。我自己也曾为凑单省两块钱花半小时比价，最后省的钱不够买瓶水。这种刻进骨子里的节俭，把时间浪费在最不值钱的地方。

另一边，有人收入不高却纵情消费：相机用两次吃灰，健身卡去两回再没去过——短暂快感后只剩账单与后悔。

为什么？因为消费主义最擅长掩盖真实代价：它喊“你值得拥有！快乐无价！”，却从不告诉你——**快乐有价，你的收入就是标尺。**

按照公式，快乐的心理单价 = 年收入 ÷ 8760 × 闲暇系数。收入越低，每小时快乐的基础价值就越低。同样一件商品，富人可能只需几十小时快乐回本，穷人却需要几百上千小时。这听起来残酷，但它反映的只是客观事实：**金钱的稀缺性不同**。同样的快乐，对穷人来说成本更高，因为钱更难赚。

对低收入者，一部手机意味几百小时生命时间。这笔消费不是在花“钱”，而是在花生命时间。这不是冷酷，是理性，阻止你用宝贵生命换取转瞬物欲——这正是饮鸩止渴。

两种极端，实为一枚硬币两面：一是“时间廉价化”（为省小钱浪费宝贵时间）；一是“消费无意识化”（为快感花掉不该花的钱）。我想要一个工具，帮人理性消费——既不贱卖时间，也不透支未来。

后来我思考了一个问题：“没苦硬吃”对不对？结论：越没钱时，节俭是无奈生存策略；经济好转后则无必要。因为你省下的几块钱，可能远低于理财收益或资产增值率。当你有一定资产，每天差几块钱的东西，完全可以无脑买，不用纠结。

这个观察，催生了“初四法则”——一个验证此想法的公式。

**这个公式恰好与你“有多少钱办多少事”的价值观自洽：**

- **公式①（时间-金钱换算器）** 让你看清：这笔消费需要花掉你多少小时的生命时间。
- **公式②（每日预算）** 让你看清：这笔消费会不会透支你的未来。

它逼每个人面对同一个问题：**你的生命时间，到底值不值得这样花？** 对富人，一小时值100元，一个包2万需要200小时，可能不值；对穷人，一小时值10元，同样2万需要2000小时，更可能不值。同一个逻辑，对所有人平等尊重。

所以，这不是悖论，而是清醒。公式不是告诉你“穷人不配快乐”，而是告诉你：

- 收入有限时，请更珍惜自己的时间；
- 收入尚可时，也请看清快乐的真实成本；
- 无论贫富，别让短暂物欲透支长久未来。

这就是“初四法则”最终想说的话。

## 二、公式介绍

这个法则是一个思考工具，它能帮你判断：买那个相机值不值？用你的时间价值去衡量，这个相机能带来多久的快乐。

你可以先算两个数字：

### 1. 你半小时的时间/快乐值多少钱？

**公式：半小时价值 = (年收入 ÷ 8760) × 闲暇系数 × 0.5**

- 8760 = 全年总小时数（365×24）
- 闲暇系数 k：反映你对闲暇/快乐的珍视程度，建议取值 2～3（0-1为不珍视，2=较珍视，3=很珍视）

### 2. 你每天可以无脑花多少钱？

**公式：每日预算 = 流动资产 × (理财收益率 - 预期增长率) ÷ 365**

- 流动资产：现金、存款、股票基金等可变现金融资产（不包括自住房、车）
- 理财收益率：可参考10年期国债（目前1.8%左右）或你自己的年化收益
- 预期增长率：0% = 保值；0.7% ≈ 2025年核心通胀；也可设为你希望的财富增速

把这两个数字记在心里，下次消费纠结时拿出来量一量。你会发现，很多纠结的事，突然就有了答案。

（注：“无脑花”是日常开销之外，允许自己奢侈一把的额度，花完不影响生活与资产增长。）

## 三、应用指引与灵活使用

### 关于必需品的说明
本法则聚焦于“可买可不买”的享乐型消费，必需品（如通讯、温饱等基础需求）不在其评判范围内——该买则买。但当你在满足基本功能的基础款与提供额外体验的升级款之间犹豫时，所支付的差价可视为“快乐投资”。此时，用快乐系数衡量：差价 ÷ 你的每小时快乐价值 = 需要多少小时快乐回本。若升级带来的显著快乐时间大于此数值，则物有所值；否则需警惕为短暂新鲜感支付过高代价。

### 灵活应用的小提示
规则为人服务，而非束缚。如果你攒了一段时间的“每日无脑预算”（例如累积30天额度）去买一件心仪已久的东西，即使一次性消耗掉一月甚至一年的额度，只要预期快乐足够强烈，完全可以这样用。之后的日子相应减少其他无脑消费即可。数字是死的，生活是活的——心里有数，灵活平衡，才是真正的理性。

## 四、如何使用这个计算器

1. 打开 `index.html` 文件（或在线访问）
2. 在“公式①”区域输入你的年收入、闲暇系数、耗时（默认0.5小时），即可算出半小时时间价值
3. 在“公式②”区域输入你的流动资产、理财收益率、预期增长率，即可算出每日可花额度
4. 下方“取值参考”卡片提供了2025年的客观数据（国债利率、CPI等）
5. 点击“初四法则的背景”可折叠展开完整的理论说明

因为我的生日，以及我悟出这个道理的那天，正好是正月初四。在这一天迎灶神的日子，我决定送给自己一份礼物：**不再为小钱纠结，也不沉迷物欲疯狂消费——那只是饮鸩止渴**，而是让时间更值钱，让消费更理性。所以取名“初四法则”，希望能帮你摆脱消费纠结，找到平衡。
